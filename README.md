# TooManyIdeasBot
 
 Create config.json:
 ```
   {
     "prefix": "!",
     "token": "<insert discord bot token here>"
   }
 ```
 
 Create data.json:
 ```
   {
     "beforeandafters": [
       {
         "plot": "<plot or question>",
         "answer": "<answer>",
         "episode": "episode this appeared on"
       },
       {
         "plot": "<plot or question>",
         "answer": "<answer>",
         "episode": "episode this appeared on"
       }
     ]
   }
```
